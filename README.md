## Digiflazz - Laravel 9 REST API

### Installation

 1. Clone this project\
 `git clone https://github.com/rosyiid1440/digiflazz-api-topup`
 2. Cd into your project folder\
 `cd digiflazz-api-topup`
 3. Install dependencies\
 `composer install`\
 Or if you want continue developing this project\
 `composer install`
 5. Copy env file\
 `cp .env.example .env`
 4. Setup your database, and digiflazz via .env file
 5. Make app key\
`php artisan key:generate`
